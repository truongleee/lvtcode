Khoá học HTML

Ngôn ngữ HTML là ngôn ngữ cơ bản nhất để tất cả các lập trình viên có thể xây dựng một website.

Khoá học CSS cơ bản

CSS cơ bản sẽ giúp bạn thiết kế giao được giao diện cơ bản của một website

Khoá học Css nâng cao

CSS nâng cao sẽ giúp bạn chuyển sâu hơn về các hiệu ứng trong thiết kế website

Khoá học JavaScript Cơ bản

JS cơ bản sẽ giúp bản nắm được kiến thức nền của ngôn ngữ này

Khoá học JavaScript nâng cao

Bạn sẽ học được gì sau khóa học HTML, CSS,JavaScript

  • Biết cách xây dựng giao diện web với HTML, CSS
  • Biết cách phân tích giao diện website
  • Biết cách đặt tên class CSS theo chuẩn BEM
  • Biết cách làm giao diện web responsive
  • HTML là gì

    HTML viết tắt của Hypertext Markup Language (tiếng việt là hay là "Ngôn ngữ Đánh dấu Siêu văn bản") là ngôn ngữ để xây dựng và cấu trúc lại các thành phần có trong website

  • CSS là gì

    CSS là chữ viết tắt của Cascading Style Sheets, nó là một ngôn ngữ được sử dụng để tìm và định dạng lại các phần tử HTML có trong webiste Sử dụng CSS giúp trang web trở nên đẹp hơn và sống động hơn với nhiều phong cách và format khác nhau

  • Học HTML và CSS có khó không?

    Bạn có thể học từ đầu mà không cần có kiến thức trước đó. Chỉ cần bạn chịu khó học tìm hiểu và thực hành nhiều ví dụ là có thể làm được Mình nghĩ HTML và CSS là dễ học vì nó khá trực quan

  • Bạn cần chuẩn bị những gì khi học HTML và CSS

    Bạn cần có máy tính kết nối internet để xem các video bài giảng này Một máy tính có cài một trình duyệt web Một phần mêm soạn thảo cơ bản như Notepad, Visual Studio Code, Nodepad ++, Sublime Text... Vậy chúng ta hãy cùng tham gia khóa học F8 HTML CSS từ đầu dành cho tất cả các bạn nhé

  • JavaScript là gì?

    JavaScript là ngôn ngữ lập trình website phổ biến hiện nay, nó được tích hợp và nhúng vào HTML giúp website trở nên sống động hơn. JavaScript đóng vai trò như là một phần của trang web, thực thi cho phép Client-side script từ phía người dùng cũng như phía máy chủ (Nodejs) tạo ra các trang web động.
